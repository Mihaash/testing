🛡️ Malware-Capy

Automated malware analysis system using YARA and CAPA for fast detection and behavioral insights.

📌 Overview

Malware-Capy is a framework that automates malware detection and analysis using signature-based scanning and capability extraction. It helps security teams quickly identify threats and understand malicious behavior.

⚙️ Key Features
  • YARA-based malware detection
  • CAPA capability analysis (behavior insights)
  • Automated analysis pipeline
  • File hash generation (MD5 / SHA256)
  • Structured report generation
🧠 Detected Capabilities
  • Keylogging behavior
  • Persistence mechanisms
  • Network communication
  • Encryption & obfuscation
⚙️ Workflow
  1. Upload suspicious file
  2. Generate file hash
  3. Run YARA scan
  4. Run CAPA analysis
  5. Aggregate results
  6. Generate final report
📊 Risk Classification
Malicious Suspicious Clean
🔐 Use Cases
  • SOC malware triage
  • Threat intelligence analysis
  • Incident response automation
  • File reputation checking
🧰 Tech Stack

Python (Flask/FastAPI), YARA, CAPA, HTML, CSS, JavaScript, Hashlib, Subprocess
