⚡ Multi-Tool Static Code Analysis Dashboard

Unified platform to scan source code using multiple security tools and visualize results in one dashboard.

🚀 Key Features
  • Upload source code (file / ZIP)
  • Runs multiple analyzers automatically
  • Aggregated results dashboard
  • Severity classification (Critical / Warning / OK)
  • Downloadable HTML & PDF reports
🛠 Integrated Tools

Bandit, Semgrep, ESLint, Checkov, Flawfinder, PMD, Gitleaks, TruffleHog, Hadolint, TFLint, Yamllint, and more.

⚙️ Workflow
  1. Upload code (file or ZIP)
  2. Auto-detect file types
  3. Run relevant security tools
  4. Classify findings by severity
  5. Display results in dashboard
📊 Severity Levels
Critical Warning OK Skipped
🔒 Security Impact
  • Detects vulnerabilities & weak cryptography
  • Finds hardcoded secrets (API keys, tokens)
  • Identifies insecure configurations
  • Improves code quality & security posture
