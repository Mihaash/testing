// OVERVIEW

A web-based URL safety analyser that lets users paste any link and instantly receive a Safe or Not Safe verdict. The tool inspects domain structure, TLD reputation, and pattern signatures to flag phishing sites, malware distributors, spam links, and suspicious domains — before the user ever clicks.

// THREAT DETECTION CATEGORIES
⚠
Phishing Sites
Detects spoofed login pages and credential-harvesting domains using pattern matching and TLD analysis.
☣
Malware / Virus
Flags URLs serving known malicious payloads, drive-by downloads, or exploit kit endpoints.
◈
Spam / Scam Links
Identifies spam domains, lottery scams, and deceptive redirect chains via structural heuristics.
⬡
Suspicious Domains
Catches newly registered, typosquatted, or algorithmically generated domains with low trust scores.
